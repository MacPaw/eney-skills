Returns the OAuth tokens that the framework resolved before your component mounted. Use this hook to make authenticated API calls to external services.

## Import

Import the hook from the Eney API package:

```typescript
import { useOAuth } from "@eney/api";
```

## Usage

Call the hook at the top of your component to get the current access token:

```typescript
const { accessToken, refreshToken } = useOAuth();
```

The `accessToken` is always a valid string — the framework guarantees that your component only renders after tokens have been resolved.

## Signature

```typescript
function useOAuth(): OAuthTokens
```

### Return value

| Property       | Type             | Description                                            |
| -------------- | ---------------- | ------------------------------------------------------ |
| `accessToken`  | `string`         | The OAuth access token for making authenticated calls  |
| `refreshToken` | `string \| null` | The refresh token, if the provider returned one        |

## Prerequisites

`useOAuth` only works inside widgets that declare an `oauth` configuration in `defineWidget`. If you call it in a widget without OAuth configured, it throws an error.

```typescript
import { defineWidget, useOAuth, Paper } from "@eney/api";
import { z } from "zod";

const schema = z.object({
  query: z.string().optional(),
});

type Props = z.infer<typeof schema>;

const MyWidget = defineWidget({
  name: "my-authenticated-widget",
  description: "Fetches data from an external API",
  schema,
  oauth: {
    clientId: "your-client-id",
    authorizeUrl: "https://provider.com/oauth/authorize",
    tokenUrl: "https://provider.com/oauth/token",
    scopes: ["read", "write"],
  },
  component: MyWidgetComponent,
});
```

## Example

This example fetches a list of repositories from GitHub using the access token:

```tsx
import { useState, useEffect } from "react";
import { Paper, useOAuth, useCloseWidget } from "@eney/api";

function MyWidgetComponent(props: Props) {
  const { accessToken } = useOAuth();
  const closeWidget = useCloseWidget();
  const [repos, setRepos] = useState<string[]>([]);

  useEffect(() => {
    fetch("https://api.github.com/user/repos", {
      headers: { Authorization: `Bearer ${accessToken}` },
    })
      .then((res) => res.json())
      .then((data) => setRepos(data.map((r: any) => r.full_name)));
  }, []);

  const markdown = repos.length
    ? repos.map((r) => `- ${r}`).join("\n")
    : "Loading repositories...";

  return <Paper markdown={markdown} />;
}
```

<Callout type="info">
  You don't need to handle the sign-in flow yourself. The framework automatically shows a "Signing in..." screen, opens the browser for authorization, and only mounts your component after tokens are available. See the [OAuth guide](/docs/oauth) for the full setup instructions, configuration reference, and how the flow works end to end.
</Callout>
