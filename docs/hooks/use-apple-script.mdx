title: useAppleScript
description: Run AppleScript via osascript from within a widget

import { Callout } from "fumadocs-ui/components/callout";

`useAppleScript` is an utility that wraps the `osascript` process into a hook-style API.

## Import

```typescript
import { useAppleScript } from "@eney/api";
```

## Usage

```typescript
const runAppleScript = useAppleScript();

const result = await runAppleScript(`
  tell application "Notes"
    get name of first note
  end tell
`);
```

## Signature

```typescript
function useAppleScript(): (script: string) => Promise<string>
```

Returns an async function. Call that function with an AppleScript string. It resolves with the trimmed stdout of `osascript`, or rejects with an error if the script fails.

## Full example — send an email

```typescript
import { useState } from "react";
import { Form, Action, ActionPanel, useCloseWidget } from "@eney/api";
import { useAppleScript } from "../shared/use-apple-script.js";

function SendMail(props: { to?: string; subject?: string; body?: string }) {
  const closeWidget = useCloseWidget();
  const runAppleScript = useAppleScript();
  const [to, setTo] = useState(props.to ?? "");
  const [subject, setSubject] = useState(props.subject ?? "");
  const [body, setBody] = useState(props.body ?? "");
  const [sending, setSending] = useState(false);

  function escape(s: string) {
    return s.replace(/\\/g, "\\\\").replace(/"/g, '\\"');
  }

  async function onSubmit() {
    setSending(true);
    try {
      await runAppleScript(`
        tell application "Mail"
          set msg to make new outgoing message with properties {
            subject: "${escape(subject)}",
            content: "${escape(body)}",
            visible: false
          }
          tell msg
            make new to recipient with properties {address: "${escape(to)}"}
          end tell
          send msg
        end tell
      `);
      closeWidget(`Email sent to ${to}`);
    } catch (error) {
      setSending(false);
    }
  }

  return (
    <Form
      actions={
        <ActionPanel>
          <Action.SubmitForm
            title={sending ? "Sending..." : "Send"}
            onSubmit={onSubmit}
            style="primary"
            isLoading={sending}
          />
        </ActionPanel>
      }
    >
      <Form.TextField name="to" label="To" value={to} onChange={setTo} />
      <Form.TextField name="subject" label="Subject" value={subject} onChange={setSubject} />
      <Form.RichTextEditor value={body} onChange={setBody} />
    </Form>
  );
}
```

<Callout type="warn">
  **Always escape user input** before interpolating into AppleScript strings. Replace `\` with `\\` and `"` with `\"`. Unescaped input can break the script or cause unexpected behavior.
</Callout>

## Error handling

`useAppleScript` rejects if `osascript` exits with a non-zero code. Always wrap calls in `try/catch`:

```typescript
try {
  const result = await runAppleScript(`tell application "Notes" to get name of first note`);
  console.log(result);
} catch (error) {
  // Handle AppleScript error
  logger.error("AppleScript failed: %s", error.message);
}
```
