The `@eney/api` package provides several hooks to use inside your widget components. Just like standard React hooks, these must be called at the top level of your function component or custom hook.

## Available hooks

<Cards>
<Card title="useLogger" href="/docs/hooks/use-logger">
    Write debug, info, warn, and error messages to the Eney log.
  </Card>
  <Card title="useCloseWidget" href="/docs/hooks/use-close-widget">
    Finalize a widget interaction and pass context back to Eney.
  </Card>
  <Card title="useAppleScript" href="/docs/hooks/use-apple-script">
    Run AppleScript via osascript and return the output as a string.
  </Card>
</Cards>

## Import

You can import most hooks directly from the Eney API package:

```typescript
import { useLogger, useCloseWidget, useAppleScript } from "@eney/api";
```
